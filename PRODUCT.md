# Register

brand

## Users

Recruiters, security hiring managers, and technical leads evaluating Ahmed Mamdouh for penetration testing, backend development, and automation engineering roles.

## Product Purpose

Premium cybersecurity portfolio communicating offensive security expertise, Python backend craft, and automation engineering — built for Cairo-based professional Ahmed Mamdouh.

## Brand Personality

Technical, precise, futuristic-minimal. Cyprus (#004741) + Sand (#F0EDE4) palette with Migha display typography and retro-futurist vector accents.

## Anti-references

Generic purple-gradient SaaS heroes, Inter + slate defaults, glassmorphism cards, gradient text, numbered section eyebrows on every block.

## Design Principles

1. Security credibility through craft, not cliché hacker aesthetics
2. Mobile experience matches desktop quality
3. Performance-first: 95+ Lighthouse targets
4. All content sourced from Ahmed Mamdouh data.pdf
5. SVG-native graphics inspired by provided mood board — never raw background images

## Accessibility & Inclusion

WCAG 2.1 AA, prefers-reduced-motion respected, keyboard navigation, visible focus states.
